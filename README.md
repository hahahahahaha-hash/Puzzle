# 🧩 经典数字/图像拼图 (Sliding Puzzle)

这是一个基于 **HTML5**, **CSS3** 和 **原生 JavaScript** 开发的益智拼图游戏。支持经典的 $3 \times 3$ 或 $4 \times 4$ 模式，挑战你的逻辑思维与空间想象力。

## ✨ 核心特性

* **动态切割**：自动将一张图片分割为 $N \times N$ 个小方块。
* **智能打乱**：采用“随机步数模拟”算法，确保生成的每一个拼图都是**有解的**。
* **平滑动画**：利用 CSS3 `transition` 实现方块滑动的视觉效果。
* **计时计步**：实时记录游戏耗时与移动步数。
* **模式切换**：支持“数字模式”与“图片模式”一键切换。

## 🕹️ 游戏指南

1. **开始游戏**：点击“打乱棋盘”按钮。
2. **移动方块**：点击空白格相邻的方块，它将自动滑动到空格位置。
3. **目标**：按照原始顺序（数字由小到大，或还原完整图片）排列所有方块。
4. **胜利**：当右下角留出空白，其余部分完整时，游戏结束！

## 📂 文件清单

* `index.html` - 包含游戏容器及控制面板。
* `style.css` - 定义拼图网格布局、方块阴影及移动动画。
* `script.js` - 处理方块索引交换、逆序数判定及胜负检测。

## 🧠 技术实现逻辑

### 1. 确保有解 (Solvability)

随机排列可能导致拼图永远无法复原。本项目通过**模拟玩家移动**的方式打乱：

> 从完整状态开始，随机选择空白格周围的方块进行 100+ 次有效移动。

### 2. 坐标计算

每个方块通过 `background-position` 属性显示图片的特定部分：


$$x_{pos} = (index \pmod{columns}) \times blockWidth$$

$$y_{pos} = \lfloor index / columns \rfloor \times blockHeight$$

## 🛠️ 如何扩展

* **更换图片**：只需修改 `style.css` 中的 `background-image` 路径。
* **调整难度**：在 `script.js` 中修改 `gridSize` 变量（如从 3 变为 5）。
* **增加音效**：在方块滑动成功时触发 `play()` 音频接口。
